>>>> Hướng dẫn chạy source code:
	1. Giải nén Restaurant.zip
	2. Tại thư mục Restaurant được giải nén mở cmd bằng cách gõ cmd tại thanh đường dẫn
	3. Chạy câu lệnh npm install để tải thư mục node_modules
	4. Sau đó để chạy project ta chạy lện npm start 
	5. Truy cập đường dẫn localhost:3000 ở trình duyệt

>>>> Truy cập các giao diện Website:
	+ Đối với trang phía khách hàng thì gõ đường dẫn: http://localhost:3000
	+ Đối với trang phía quản lý nhà hàng thì thay bằng: http://localhost:3000/comnha
